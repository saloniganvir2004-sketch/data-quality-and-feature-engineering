# Data Quality and Feature Engineering

This repository demonstrates practical techniques for preparing raw datasets for machine learning by focusing on data quality assessment and feature engineering.

The project uses a real-world educational dataset and walks through industry-relevant steps such as missing value handling, outlier detection, feature creation, scaling, and encoding.
